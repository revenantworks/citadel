Naming — quick rules
  • Marketplace: revenantworks/citadel — installs read foundation@revenant
  • Org: revenantworks · repos lowercase, revenant-<thing> · skills revenant-<pack>-<skill> · releases vX.Y.Z
  • Packs: named for places in the Revenant world; each pack owns a member motif — -smith is foundation's. On file: forge/-wright · crypt/-keeper · threshold/-warden · wake/-watcher · vault/-locker
  • Marks: revenant lowercase in marks, "Revenant" in prose; the handle carries "works," the mark never does
Voice

Revenant — the machine that came back

Calm, exact, inevitable. Short declaratives; no exclamation marks, no hype emoji. Docs plain · releases dry + one flavor line max · social minimal, lowercase fine.

Releases: "Back online." · Social: "— RVNT"
Wordmarks & logo rules
  • revenant▍ — lowercase mono, cursor in threshold-blue (blinks on web). Monogram: RVNT, or an R looping into a return arrow.
  • Clearspace: one cursor-block width on all sides. Minimums: monogram 24px, wordmark 120px.
  • Never: stretch or recolor outside tokens · shadows or gradients (glow is the only light effect) · magenta + green on one mark · marks over busy imagery without a void scrim.
Motion · imagery · applications
Motion
  • House: cursor blinks ~1s; everything else fades 150–250ms. Nothing bounces.
  • Glitch bursts ≤500ms, one per composition; reduced-motion → static.
Imagery
  • Revenant: abstract grids, particles, signal traces. No faces, no stock.
Applications
  • Overlays: LIVE = ember; text bone-white.
  • Avatars: monograms only. Layout: 8px grid, 14px radius.
Accessibility
  • Bone on void ≈ 16:1 — body always passes.
  • Accents = large text / graphics only; never accent-on-accent.
  • Focus rings 2px threshold-blue.
