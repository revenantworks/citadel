# Voices — Volatile *(single update surface)*

> **Last saved: 2026-07-13.** The only file "commsmith voice" rewrites. Saved voices persist here between sessions where file tools exist; where they don't, a voice profile is pasted per session and consumed the same way.

## Voice profile format

| Field | Carries |
|---|---|
| name | How the voice is invoked ("work-me", "channel-voice") |
| register | Baseline tone in one line |
| cadence | Sentence rhythm — short/punchy, flowing, clipped |
| lexicon — do | Words and moves this voice uses |
| lexicon — don't | Words and moves this voice never uses |
| sign-off | How messages close, per channel where it differs |
